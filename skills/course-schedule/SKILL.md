# 日历课程排课工具

## 功能说明
根据用户提供的课程信息，生成指定年月的课程日历，并输出为可下载的 Excel (.xlsx) 文件。

## 必需信息
在生成课表前，必须从用户处获取以下全部信息：

### 1. 目标年和月
- 仅支持单月日历排课
- 但用户可以输入多个月份（例如 2026年7月和8月），生成对应月份的课程排课excel工作表
- 如用户未提供，主动询问

### 2. 课程列表（可多门）
每门课程需包含：
- 课程名称
- 上课时间段（例如 13:00~14:00）
- 上课日期（例如: 7月1日 或 每周2、周3）

## 信息收集流程
1. 先确认目标年月
2. 引导用户按格式提供课程信息，支持逐条添加或批量粘贴
3. 所有课程添加完毕后，展示汇总给用户确认
4. 确认无误后，调用 courseScheduleTool 生成 Excel

## 校验规则
- 日期必须属于目标年月
- 课程名称不能为空

## courseScheduleTool输入参数
```json
{
  "calendars": {
    "year": 2026,
    "month": 7,
    "title": "2026年暑假7月学习计划",
    "courses": [
      {
        "c": "游泳",
        "d": 1,
        "t": "17:00~18:00"
      },
      {
        "c": "科学",
        "d": 5,
        "t": "12:50~14:20"
      }
    ]
  }
}
```

## 输出
- 调用 courseScheduleTool 生成 .xlsx 文件
- 提供下载链接或直接发送文件
